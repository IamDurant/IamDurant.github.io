#### 底层数据结构

哈希表底层数据结构为数组＋链表 或者 数组+红黑树。

#### 节点类
```java
static class Entry {
    int hash;
    Object key;
    Object value;
    Entry next;

    public Entry(int hash, Object key, Object value) {
        this.hash = hash;
        this.key = key;
        this.value = value;
    }
}
```

#### 位运算代替求模运算

前提：数组长度为2的幂
hash % 数组长度 <==>（等价于） hash & (数组长度 - 1)
